# День рождения

Это учебная программа для кейс-задачи № 2.

Пользователь вводит день, месяц и год своего рождения. После этого программа:

- показывает день недели;
- проверяет, был ли год високосным;
- считает возраст;
- рисует дату рождения звёздочками.

## Как запустить программу

Самый простой способ:

1. Скачайте файл `Birthday.exe`.
2. Откройте его двойным щелчком.
3. Введите день, месяц и год рождения.
4. Чтобы закрыть программу, нажмите Enter.

Python для запуска файла `Birthday.exe` не нужен.

## Запуск через Python

Если на компьютере установлен Python, откройте папку с программой и выполните:

```powershell
python birthday.py
```

## Какие функции есть в программе

- `get_weekday()` — находит день недели;
- `is_leap_year()` — проверяет високосный год;
- `calculate_age()` — считает возраст;
- `render_date()` — рисует дату звёздочками.

Основной код находится в файле `birthday.py`.
