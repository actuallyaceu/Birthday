# Стилистическое преобразование чисел

Консольная программа запрашивает дату рождения пользователя и выводит:

- день недели, на который пришлась дата рождения;
- был ли год рождения високосным;
- текущий возраст пользователя в полных годах;
- дату рождения в формате `дд мм гггг`, нарисованную звёздочками.

## Запуск

Требуется Python 3.10 или новее.

```powershell
python birthday.py
```

Пример ввода:

```text
Введите день рождения: 3
Введите месяц рождения: 2
Введите год рождения: 2001
```

## Проверка тестов

```powershell
python -m unittest -v
```

## Запуск готового EXE

Файл `Birthday.exe` можно запустить двойным щелчком. Python для этого не нужен.
После вывода результата программа ожидает нажатия Enter, чтобы окно консоли не
закрылось автоматически.
